# Institute of Digital Risk (IDR) Landing Page

A professional, single-page landing page built for the **Institute of Digital Risk**. This project focuses on a "Fintech" minimalist aesthetic to communicate trust, institutional authority, and digital resilience.

## Live Demo
[View the Live Site Here](https://purvashi04.github.io/insti_digi_risk/)

## Design Strategy
- **Identity:** Utilizes an isometric cube logo representing structural resilience.
- **Palette:** Dark Violet (#4B0082) for authority and Signal Yellow (#FFD700) for risk-awareness.
- **Typography:** The 'Inter' typeface for a modern, geometric tech feel.

## Technical Features
- **Semantic HTML5:** Clean structure using `<nav>`, `<header>`, `<main>`, and `<section>`.
- **Responsive Design:** Built with **CSS Grid** and **Flexbox** for mobile and desktop compatibility.
- **Smooth Interaction:** Sticky navigation bar with vanilla JS smooth scrolling.
- **Accessibility:** High-contrast colors and readable typography.

## Project Structure
- `index.html` - Core structure and content.
- `style.css` - Custom fintech styling.
- `script.js` - Interactivity and form handling.
- `/assets` - SVG Logo variants.
